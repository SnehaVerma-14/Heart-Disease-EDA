# ❤️ Heart Disease Risk Analysis using Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the UCI Heart Disease dataset to understand factors associated with heart disease. The analysis includes data cleaning, visualization, outlier detection, and correlation analysis.

## 🎯 Objectives
- Understand the distribution of heart disease cases.
- Analyze the impact of age and gender.
- Study cholesterol and blood pressure patterns.
- Examine maximum heart rate and chest pain type.
- Identify relationships between variables using a correlation heatmap.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab / Jupyter Notebook

## 📊 EDA Techniques
- Data Cleaning
- Missing Value Handling
- Histograms
- Box Plots
- Count Plots
- Correlation Heatmap
- Outlier Detection
- Demographic Analysis

## 📂 Dataset
This project uses the UCI Heart Disease dataset.

## 🚀 Key Insights
- Age is an important factor associated with heart disease.
- Male patients show a higher occurrence of heart disease in this dataset.
- Chest pain type provides valuable information about disease status.
- Cholesterol, blood pressure, and maximum heart rate show meaningful patterns.
- Correlation analysis helps identify relationships among clinical features.
